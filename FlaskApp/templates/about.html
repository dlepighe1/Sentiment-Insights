About This Project

This web application delivers advanced sentiment analysis for both single text inputs and bulk documents. Designed to help users extract meaningful insights, it supports multiple powerful machine learning models including Logistic Regression, Random Forest, and a state-of-the-art large language model (Meta Llama 3.3).

How to Use This Application

1. Text Prediction

Simply type or paste your text into the input form and submit to get an instant sentiment prediction. The results include:

  • Predicted Sentiment: The sentiment label assigned to your input (Positive, Neutral, or Negative).
  • Probability Scores: Percentages reflecting the model's confidence for each sentiment class.
  • Top Contributing Words: Highlights important words and their quantitative contributions to the final prediction.

For the Meta Llama model, an additional Explanation section provides a short reasoning behind the prediction, showcasing the interpretability power of large language models.

2. Bulk Prediction

Upload or drag and drop a file in CSV or TXT format containing multiple texts for batch sentiment analysis. The application:

  • Reads each line or row of text from your file.
  • Predicts the sentiment for every entry.
  • Returns the results including an added sentiment column and a sentiment score for each line.

Note: The sentiment score corresponds to the highest sentiment probability for that text. For example, if the predicted label is "Positive" with a score of 45%, the other sentiments (Neutral, Negative) have lower probabilities such that all scores sum to approximately 100%. An illustrative split might look like 45% Positive, 40% Negative, and 15% Neutral.

3. Web Scraping (Coming Soon)

We're excited to introduce a web scraping feature soon that will allow you to extract text data directly from websites for seamless sentiment analysis at scale. Stay tuned!

Our Goal

We aim to provide an accessible and interpretable tool that empowers individuals and organizations to understand sentiment and opinion trends in text data efficiently, whether for market research, social media monitoring, or customer feedback analysis.
